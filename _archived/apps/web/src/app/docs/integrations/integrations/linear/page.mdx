## server ‚Äî api route

`toLinearIssue` gives you a title, markdown description with placeholder tokens, priority, label suggestions, and files. the server uploads each file to linear, replaces the placeholder in the description with the returned asset url, and then creates the issue.

```ts
// app/api/feedback/route.ts
import { parseFeedback } from "remediate/server";
import { toLinearIssue } from "remediate/linear";
import { LinearClient } from "@linear/sdk";

const linear = new LinearClient({ apiKey: process.env.LINEAR_API_KEY! });
const TEAM_ID = process.env.LINEAR_TEAM_ID!;

export async function POST(req: Request) {
  const feedback = await parseFeedback(req);
  const issue = toLinearIssue(feedback);

  // upload each file to linear and swap its placeholder for the asset url
  let description = issue.description;

  for (const file of issue.files) {
    const upload = await linear.fileUpload(
      file.content.type,
      file.filename,
      file.content.size,
    );

    if (upload.success && upload.uploadFile) {
      const { uploadUrl, assetUrl, headers } = upload.uploadFile;
      const buffer = Buffer.from(await file.content.arrayBuffer());

      await fetch(uploadUrl, {
        method: "PUT",
        headers: {
          ...headers.reduce(
            (acc, { key, value }) => ({ ...acc, [key]: value }),
            {} as Record<string, string>,
          ),
          "Content-Type": file.content.type,
          "Cache-Control": "public, max-age=31536000",
        },
        body: buffer,
      });

      const isImage = file.filename.endsWith(".png") || file.filename.endsWith(".jpg");
      const markdown = isImage
        ? `![${file.filename}](${assetUrl})`
        : `[${file.filename}](${assetUrl})`;
      description = description.replace(file.placeholder, markdown);
    }
  }

  const created = await linear.createIssue({
    teamId: TEAM_ID,
    title: issue.title,
    description,
    priority: issue.priority,
  });

  const createdIssue = await created.issue;
  return Response.json({
    ok: true,
    id: feedback.submission.id,
    linearIssue: createdIssue?.identifier,
  });
}
```

## output shape

- `title` ‚Äî auto-derived from the first text note, annotation, or item caption
- `description` ‚Äî markdown body with `{{screenshot-‚Ä¶}}`, `{{recording-‚Ä¶}}`, and `{{voice-‚Ä¶}}` placeholders you replace after upload
- `priority` ‚Äî linear priority integer derived from the highest-priority annotation (`0`=none, `1`=urgent, `2`=high, `3`=medium, `4`=low)
- `labelNames` ‚Äî suggested label names based on priority tags
- `files` ‚Äî blob + placeholder pairs to upload and embed
